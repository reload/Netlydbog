
-- Summary -- 

The "facebook social comments" integrate the facebook's social comments plugin in Drupal
(see http://developers.facebook.com/docs/reference/plugins/comments). 

It has the following features:

* Allows site administrators to have facebook users commenting on their Drupal site posts
* Administrator can select which content type can use facebook comment widget
  NOTE: If you specifiy a certain content type to use fb_comments, then drupal comments 
  settings for that content type will be ignored. The fb_comments will hijack the node comments!
* Allows customization of the widget

-- Requirements --

Requires the "facebook social" module which is included in this package


-- Installation -- 

* Install as usual, see http://drupal.org/node/70151 for further information.
